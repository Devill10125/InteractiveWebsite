# InteractiveWebsite
Pyramid Explorer is an interactive site that focuses on user accessibility and narrative driven adventure.

The project combines branching narrative gameplay with keyboard-only controls, audio narration and high-contrast visual feedback designed to improve accessibility for players with visual or hearing impairments.

#Features
Interactive branching story with multiple endings
Keyboard-only navigation
Audio narration
Replay narration functionality
Color-assisted choice confirmation system
Atmospheric ambient audio

#Controls
SPACE — Start 
A — Select left choice
D — Select right choice
R — Replay narration audio

#Technologies Used
HTML5
CSS3
JavaScript
